## セキュリティポリシー（Security Policy）

**zodapp** と利用者の安全にご協力いただきありがとうございます。

## サポート対象バージョン（Supported versions）

本リポジトリ内の各パッケージについて、セキュリティ更新は原則として **最新のリリース版** を対象に提供します。

## 脆弱性の報告（Reporting a vulnerability）

セキュリティ上の問題を **公開の GitHub Issue で報告しないでください**。

代わりに、次のいずれかを利用してください:

- GitHub Security Advisories（推奨）
  - リポジトリの「Security」タブから非公開レポートを作成してください
- Advisories が利用できない場合は、GitHub 上でメンテナへ非公開で連絡してください

報告時には、可能な範囲で以下を含めてください:

- 問題の内容と影響の説明
- 再現手順（可能であれば PoC）
- 影響を受けるパッケージとバージョン
- 想定される回避策/緩和策

## 開示（Disclosure）

受領後、調査して修正に取り組みます。必要に応じて、報告者と開示タイムラインを調整します。
