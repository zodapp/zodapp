## Security Policy

Thanks for helping keep **zodapp** and its users safe.

## Supported versions

Security updates are provided for the latest released versions of packages in this repository.

## Reporting a vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, use one of the following:

- GitHub Security Advisories (preferred)
  - Create a private report in the repository’s “Security” tab
- If you cannot use advisories, contact the maintainers privately via GitHub

When reporting, please include:

- A clear description of the issue and impact
- Steps to reproduce (proof-of-concept if possible)
- Affected package(s) and version(s)
- Any suggested mitigation

## Disclosure

We will acknowledge receipt, investigate, and work on a fix. We may coordinate a disclosure timeline with the reporter when appropriate.
